# Elegant Watches


## User Stories:  

1. As a user, I want to create a new account. 

2. As a user, I want to log in to the website. 

3. As a user, I want to log out from the website. 

4. As a user, I want to add watches to display on the website. 

5. As a user, I want to see added watches with their descriptions.
